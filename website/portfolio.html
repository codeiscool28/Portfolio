Projects

Countdown Timer

This project is a countdown timer for events, created using HTML and JavaScript. It prompts the user to input the name and date of an event, then calculates and displays the time remaining until the event in days, hours, minutes, and seconds.

Note: This was a hypothetical project for university and school.

Shopping Cart System

This project is a web-based shopping cart system. Users can select products, specify quantities, and choose a shipping method. The system validates inputs, displays the shopping records in a table, and calculates the total cost including a delivery fee if applicable.

Note: This was a hypothetical project for university and school.

Australian Births Registered Calculator

This project collects data on births registered in different Australian states for 2018 and 2019. It validates user inputs, calculates changes in the number of births between the two years, and displays the data in a table along with statistical summaries.

Note: This was a hypothetical project for university and school.

HomewareCity Shopping Cart System

This project is an interactive shopping cart system for HomewareCity, a family business in Toowoomba. The system allows users to add new shopping records, display details, calculate total costs, search, and sort through records. It enhances the shopping experience by providing an online shopping facility and is implemented in JavaScript to be run in the Firefox browser.

Note: This was a hypothetical project for university and school.

Text-Based Adventure Game

This project is a text-based adventure game developed in Python, challenging players to navigate through a dark forest with various paths, making strategic choices that influence the outcome of their journey. It features dynamic story paths and engages players with interactive decision-making.

Note: This project demonstrates programming logic, user interaction, and narrative development in a simple command-line application.
